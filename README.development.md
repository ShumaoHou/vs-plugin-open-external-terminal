# Installation and Development / 安装与开发

## Installation / 安装

Install the packaged VSIX file in VS Code with `Extensions: Install from VSIX...`.

在 VS Code 中使用 `Extensions: Install from VSIX...` 安装打包好的 VSIX 文件。

## Development / 开发

```bash
npm install
npm run compile
```

Open this folder in VS Code and press `F5` to launch an Extension Development Host.

使用 VS Code 打开本目录，按 `F5` 启动 Extension Development Host 进行调试。
